# save_as!Sonic: Emerald Quest (SA!S:EQ)

A remake of Sonic: Emerald Quest, a scrapped STJr game from the late 90's in the engine of [Sonic Robo Blast 2](https://srb2.org/), version 2.1. 

Features Action RPG combat and is planned to have an expansive world with three stories and a last story.

**Currently very unfinished, anyone is welcome to contribute**

## Dependencies
- NASM (x86 builds only)
- SDL2 (Linux/OS X only)
- SDL2-Mixer (Linux/OS X only)
- libupnp (Linux/OS X only)
- libgme (Linux/OS X only)

## Compiling
Compiling should largely be the same as it is for Sonic Robo Blast 2.
See [SRB2 Wiki/Source code compiling](http://wiki.srb2.org/wiki/Source_code_compiling).
